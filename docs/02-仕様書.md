## 1. システム概要

Codex Studio は、ローカルワークスペースを対象とした AI 支援開発環境である。ユーザーはエディタ、チャット、エージェントを同一 UI 上で利用し、LLM がツールを通じてコードベースを読み書き・実行する。

## 4. 外部インターフェース

### 4.1 LLM プロバイダ

| プロバイダ | プロトコル | 用途 |
|------------|------------|------|
| OpenAI | REST / SSE | Chat, Agent |
| Anthropic | REST / SSE | Chat, Agent |
| Ollama | REST | ローカル推論 |
| カスタム | OpenAI 互換 API | 自社ゲートウェイ |

### 4.2 MCP（Model Context Protocol）

- Phase 2 で stdio / SSE トランスポート対応
- ツール・リソース・プロンプトの discovery

### 4.3 Git

- 既存 Git CLI 連携（status, diff, commit は Shell ツール経由）
- ネイティブ libgit2 連携は Phase 2 以降

## 5. データ要件

### 5.1 永続化データ

| データ | 保存先 | 保持期間 |
|--------|--------|----------|
| 会話履歴 | SQLite | ユーザー設定（デフォルト無制限） |
| 索引メタデータ | SQLite + ファイルキャッシュ | ワークスペース存続中 |
| 設定 | JSON（暗号化フィールドあり） | 永続 |
| 監査ログ | JSONL | 90日（デフォルト） |

### 5.2 個人情報・機密

- コード内容はデフォルトで外部に送信しない（ユーザー明示操作時のみ）
- テレメトリはオプトイン

## 6. UI 仕様（概要）

### 6.1 レイアウト

```
┌─────────────────────────────────────────────────────────────┐
│ Menu / Command Palette                                      │
├──────────┬──────────────────────────────┬───────────────────┤
│ Activity │        Editor Area           │   AI Panel        │
│  Bar     │  (Tabs + Monaco Editor)      │  Chat / Agent     │
│          │                              │  Tool Log         │
│ Explorer │                              │                   │
│ Search   │                              │                   │
│ Git      │                              │                   │
├──────────┴──────────────────────────────┴───────────────────┤
│ Status Bar（モデル、索引状態、Git ブランチ）                    │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 主要画面遷移

1. 起動 → ウェルカム → ワークスペース選択
2. エディタ ↔ AI パネル（`Cmd+L` トグル）
3. Agent 実行中 → ツールログ展開 → Diff 確認 → 適用/却下

## 7. 受入基準（MVP）

- [ ] 10,000 ファイル規模のリポジトリを開き、30s 以内に索引完了
- [ ] Agent が「README に Installation セクションを追加」タスクを完遂
- [ ] Shell 実行前に確認ダイアログが表示される
- [ ] 会話を閉じて再起動後、履歴が復元される
- [ ] OpenAI / Anthropic / Ollama のいずれかで応答可能
- [ ] macOS / Windows でインストーラーから起動可能

## 8. 用語集

| 用語 | 定義 |
|------|------|
| Agent | ツールを自律的に呼び出す LLM 実行モード |
| Tool | LLM が呼び出せる関数（Read, Shell 等） |
| Session | 単一の会話スレッド |
| Rules | プロジェクト/global の LLM 指示 |
| MCP | Model Context Protocol（Anthropic 提唱の拡張規約） |
| TTFT | Time To First Token |
